# Eyuxia Homepage

Static personal homepage for showcasing smart hardware side projects.

## Files

- `index.html`: homepage entry
- `styles.css`: page styles
- `assets/esp32-stock-ticker-demo.jpg`: hero and project image

## Local Preview

This is a static site. You can open `index.html` directly in a browser.

## Deployment Suggestion

Recommended deployment:

- Cloudflare Pages
- custom domain: `eyuxia.xyz`

Suggested structure:

- homepage root -> `eyuxia.xyz`
- stock proxy stays on `stock.eyuxia.xyz`
